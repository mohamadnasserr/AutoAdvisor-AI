# Data

## Directories

- `data/raw`: immutable source datasets.
- `data/processed`: normalized and enriched datasets.
- `data/seed`: curated demo inventory, dealerships, reviews, and documents.
- `chroma_db`: local RAG vector index.
- `models`: trained model artifacts.
- `uploads`: safe, accepted user uploads.

## Policy

Use public datasets, official APIs, approved listing APIs, curated content, and
seeded demo data. Record source and license information for every dataset. Do
not ingest restricted marketplace data without permission.
