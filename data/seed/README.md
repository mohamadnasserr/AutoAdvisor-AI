# Demo Inventory Seed

`cars.csv` is a curated AutoAdvisor AI demo inventory for PostgreSQL. It is not
live-scraped inventory and does not claim real-time availability.

The seed contains fictional demo dealership contacts and listing statuses for
search, recommendations, comparisons, and dealer-inquiry demonstrations. A
production system should replace this seed with approved dealer feeds or
marketplace partnerships.

Image URLs use representative vehicle photos from Wikimedia Commons for demo
presentation. They are not verified photos of the seeded listings.
